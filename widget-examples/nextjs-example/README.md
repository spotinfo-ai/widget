```npm install```

```npn run dev```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.



Pushing a new tag
- Create a tag with message
```git tag -a v1.0.6 -m "new dist file with external reach dependency"```

- Push to git
```git push origin v1.0.6```
